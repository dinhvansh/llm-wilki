# Andros Vector Compass

Andros Vector Compass is an internal codename for a knowledge retrieval policy used in the AI-native wiki platform.

## Core Rule

The Vector Compass policy says every high-risk answer must include at least 2 source citations and 1 related page link before it can be shown to end users.

## Operations Note

The current staging rollout owner for Vector Compass is Minh Tran. The scheduled review cadence is every 14 days.

## Local Model Note

For local development, the preferred model connection is an OpenAI-compatible endpoint or Ollama exposed from the host machine into Docker.
